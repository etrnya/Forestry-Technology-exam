# RPD (需求規劃與系統規格定義書) - 林業技術歷屆申論試題學習與評估系統

本文件定義林業技術歷屆申論試題學習與評估系統的需求範疇、系統架構、資料模型與開發里程碑。

---

## 🎯 1. 專案背景與系統目標

### 1.1 專案背景與痛點
林業技術類科的高普考專業科目（森林生態學、育林學、森林經營學、林產學）在題型上採**全申論題**（全問答題型）。傳統以「選擇題單擊作答」為核心的練習系統（如護理師、驗光師模擬測驗系統）無法直接套用。
考生在準備申論題時，最常面臨的痛點包括：
* 缺乏系統化的歷屆申論題庫分類與篩選工具。
* 撰寫擬答大綱後，無法確認是否涵蓋「核心關鍵字」與「評分要點」。
* 缺乏即時的答題回饋與邏輯架構評析。

### 1.2 系統目標
本專案旨在建立一個**歷屆申論題庫瀏覽與 AI 智慧答題評估系統**。考生可於系統中：
1. 依照年份與科目篩選並瀏覽歷屆申論試題。
2. 在輸入框中填寫「擬答大綱」或「詳細解答」。
3. 調用本機 Gemini API，自動比對該題的標準評分要點，並對擬答進行核心點覆蓋率、結構邏輯與論述深度的綜合給分與回饋。

## 🎨 2. UI 設計與視覺規範 (SaaS Glow - Forest Green Style)

為了營造林業與森林保育的學科氛圍，本系統採用**自然森林綠**主題色調：
* **背景色**：`#09180f` (深林綠偏黑) 到 `#030712` 的漸層。
* **卡片色**：`rgba(20, 35, 25, 0.7)` 搭配微弱的綠色磨砂玻璃邊框。
* **強調色**：`#22c55e` (嫩葉綠) / `#10b981` (薄荷綠)。
* **文字色**：主文字為 `#f0fdf4` (淡綠白)，次要文字為 `#86efac` (淡綠灰色)。

## 🛠️ 3. 系統模組設計

### 3.1 爬蟲模組 (`download_moex_essays.py`)
* **目標**：從考選部「考畢試題查詢平臺」下載 110~114 年「林業技術」高考三級與普通考試的專業考科試題 PDF。
* **學科代碼對照**：
  * `forest-ecology` (森林生態學，包含保育)
  * `silviculture` (育林學)
  * `forest-management` (森林經營學)
  * `forest-products` (林產學)

### 3.2 結構化解析與 LLM 大綱生成模組 (`parse_essays.py`)
* **試題解析**：使用 `pdfplumber` 讀取 PDF 文字，按題號將申論題切分成獨立結構。
* **大綱生成**：使用 Gemini API 對每道申論題進行前置分析，自動產出：
  * **參考答題大綱** (建議寫作步驟與子標題結構)。
  * **評分關鍵字** (必須提及的名詞或概念)。
  * **難易度與出題頻率**。
* **資料輸出格式**：
  * `questions_db.js`：包含試題基本欄位與大綱。
  * `answers_db.js`：包含隱藏的核心關鍵字與標準詳解要點。

### 3.3 前端學習門戶 (`study_portal.html`)
* **Bento 資訊儀表板**：顯示已練習題數、申論平均得分、各科目練習進度。
* **申論練習區**：
  * 左側：展示申論題幹、子問題及「AI 提示大綱」（可點擊展開/隱藏）。
  * 右側：提供富文本或純文字「擬答輸入區」。
  * 底部：實作「提交 AI 評估」與「保存本地草稿」按鈕。
* **AI 答題審核面板**：提交後，展示詳細的評估報告，包括：
  * **評分** (1-100 分)。
  * **核心點覆蓋率分析** (比對關鍵字與論述要點)。
  * **優點與改進建議**。

## 📂 4. 資料庫 Schema 定義

### `questions_db.js` 格式：
```javascript
const questions_db = [
  {
    "id": "moex-114-1-forest-ecology-01",
    "year": 114,
    "session": 1,
    "subject": "森林生態學",
    "subject_code": "forest-ecology",
    "category": "高考三級",
    "q_num": 1,
    "question": "試述森林溫室效應之成因，並說明森林如何透過碳吸存（Carbon sequestration）功能減緩全球暖化之效應？(25分)",
    "tags": ["溫室效應", "碳吸存", "全球暖化"],
    "difficulty": "中等",
    "ai_hints": [
      "第一部分：溫室效應成因（溫室氣體增加、熱能滯留）。",
      "第二部分：碳吸存機制（光合作用固定二氧化碳、森林蓄積量）。",
      "第三部分：減緩暖化的森林經營對策（育林、避免毀林）。"
    ]
  }
];
```

```javascript

      "3. 森林生態系碳循環：地上部生物量與地下部土壤碳庫的作用。",

    ]
  }
};
```
